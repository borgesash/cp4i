This folder contains sample ACE BAR files 

- HttpEchoApp: This is simple flow that responds with server timestamp
  - learn how to deploy the BAR file by using the [App Connect Dashboard](https://github.com/borgesash/cp4i/blob/main/ace/deploy-sample-ace-bar-using-ui.md).
  - learn how to deploy the BAR file by using the [CLI](https://github.com/borgesash/cp4i/blob/main/ace/deploy-sample-ace-bar-using-cli.md)
- EchoServiceLoadBalancing:  This is simple HTTP flow that responds with a message to say which pod the flow ran in. This can be deployed to ACE servers with multiple replicas to demo load balancing.

How to deploy BAR files?
- [Creating an integration runtime to run your BAR file resources](https://www.ibm.com/docs/en/app-connect/13.0.x?topic=ddtiir-creating-integration-runtime-run-your-bar-file-resources)
